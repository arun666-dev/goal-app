
# GoalSetter App.

User can register/login and perform CRUD operation on individual goals


## Run Locally

Clone the project

```bash
  git clone git@github.com:arun666-dev/goal-app.git
```

Go to the project directory

```bash
  cd goal-app
```

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```
### Install dependencies
```
npm run server
```
